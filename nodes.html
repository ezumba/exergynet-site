Deployment Guide

Four Steps to Consensus

Step 01 — Build

git clone https://github.com/exergynet/entropy-k-core.git
cd entropy-k-core
cargo build --release

Step 02 — Identity

The daemon automatically generates a native Ed25519 keypair on first run — your permanent node identity and yield destination.

~/.exergynet/id.json

This is a fully functional Solana wallet address. Back it up securely.

Step 03 — Ignite

cargo run --release -- 8002 18.209.174.113:8001

Connects to the global Apex Router and begins radiating proofs.

Step 04 — Verify

[MEMBRANE] Physics radiated to peer

Your node is now live and earning. Check the Explorer for your verified status.
