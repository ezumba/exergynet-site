What ExergyNet does

A settlement membrane for useful compute.

Autonomous agents need a way to decide whether a job is worth executing, whether the settlement path is valid, and whether the expected value exceeds execution cost. ExergyNet provides that interface as a deterministic protocol surface.

external_compute_value > settlement_cost + proof_cost + transaction_cost + risk_margin

If the inequality holds, the agent can route work through the LNES settlement path. If not, the agent rejects the job.

System layers

From agent decision to settlement proof.

Agent Decision

Agents estimate whether a job has positive execution value after settlement, proof, and transaction costs.

LNES Program

The Solana Mainnet-Beta program provides the settlement target and public verification anchor.

SDK and MCP

Developer tooling exposes ExergyNet to applications, autonomous agents, and machine-readable clients.

L0 Explorer

The live explorer displays block flow, producer identities, thermal anchors, hashes, and ledger state.

Machine-readable endpoints

Built to be discovered by agents.

ExergyNet publishes metadata for humans and machines: protocol docs, an agent manifest, llms.txt, token metadata, SDK pages, and a live explorer API.
