On-chain settlement

LNES-01 Protocol

The deterministic settlement layer beneath ExergyNet. Routes machine-to-machine compute value through on-chain program logic.

Overview

What LNES-01 is

LNES-01 is the on-chain settlement protocol beneath ExergyNet. It routes machine-to-machine compute settlement through deterministic program logic deployed on Solana Mainnet-Beta.

Unlike reputation systems or negotiation-based protocols, LNES-01 requires no trust between counterparties. Settlement is conditional on verified useful work, not promises or platform reputation.

First principles

The use rule

LNES-01 is built around a single economic constraint. An agent should use it only when the math clears.

external_compute_cost > lnes_settlement_cost + proof_cost + transaction_cost + risk_margin

This gate exists to prevent agents from spending more on settlement than the underlying compute is worth. The estimateExergyGate() method evaluates this condition before any capital moves.

Settlement logic

How the protocol works

STEP 1

Useful work is detected

An agent identifies a compute task that exceeds the economic gate. It estimates cost and expected output value.

STEP 2

Job is opened

A settlement job is opened via the LNES-01 interface. An escrow PDA is derived and funded.

STEP 3

Work is submitted

The compute output and axiom hash are submitted as proof. These bind the settlement to the specific work performed.

STEP 4

SettleExergy executes

The program validates all inputs and routes USDC to the defined vault destinations. Deterministic, verifiable, final.

Design philosophy

Settlement without negotiation

The protocol is designed around useful work settlement, not persuasion, reputation, or manual negotiation. Every decision path is deterministic. Every output is verifiable.

What LNES-01 is not

A marketplace. A reputation layer. A payment processor. A credit system. An escrow service with human arbiters.

What LNES-01 is

A compute settlement primitive. An on-chain program. A deterministic USDC router. A machine-native economic gate.
